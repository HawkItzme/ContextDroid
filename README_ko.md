# ContextDroid

ContextDroid는 RTK에서 파생된 독립 유지 관리 Android 제품입니다. 출력을 줄이기 전에
원본 stdout/stderr를 저장하며, 파서 신뢰도가 낮으면 원본 출력을 그대로 반환합니다.
`rtk-ai`와 제휴하거나 보증받은 제품이 아닙니다.

전체 한국어 번역은 아직 준비 중입니다. [README.md](README.md),
[UPSTREAM.md](UPSTREAM.md), [docs/SAFETY_CONTRACT.md](docs/SAFETY_CONTRACT.md)를
참조하십시오. 기존 RTK 설치 지침을 ContextDroid에 사용하지 마십시오.
