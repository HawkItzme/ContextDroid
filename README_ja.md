# ContextDroid

ContextDroid は RTK から派生した独立管理の Android 向け製品です。出力を削減する前に
stdout/stderr をそのまま保存し、解析の信頼度が低い場合は元の出力を返します。
`rtk-ai` との提携・承認関係はありません。

完全な日本語訳は準備中です。[README.md](README.md)、[UPSTREAM.md](UPSTREAM.md)、
[docs/SAFETY_CONTRACT.md](docs/SAFETY_CONTRACT.md) を参照してください。旧 RTK の
インストール手順を ContextDroid に使用しないでください。
