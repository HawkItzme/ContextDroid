# Support

Use GitHub Issues for reproducible ContextDroid defects and feature requests. Search existing
issues first. Include the version, profile, output mode, platform/toolchain, exact redacted
command, exit code or signal, and whether raw recovery succeeded.

Never post unredacted raw build or Logcat output. It may contain secrets, private paths, package
names, device identifiers, or personal data. Report vulnerabilities through GitHub private
vulnerability reporting as described in [SECURITY.md](SECURITY.md).

The alpha is best-effort community software. There is no guaranteed response time or commercial
support channel.
