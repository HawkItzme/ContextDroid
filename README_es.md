# ContextDroid

ContextDroid es un producto Android independiente derivado de RTK. Guarda stdout y stderr
sin modificar antes de reducir la salida y vuelve a la salida original cuando la confianza
del analizador es baja. No está afiliado ni respaldado por `rtk-ai`.

La traducción completa aún no está disponible. Consulte [README.md](README.md),
[UPSTREAM.md](UPSTREAM.md) y [docs/SAFETY_CONTRACT.md](docs/SAFETY_CONTRACT.md). No use
las instrucciones antiguas de instalación de RTK para ContextDroid.
