# ContextDroid

ContextDroid est un produit Android indépendant dérivé de RTK. Il conserve les sorties
stdout/stderr brutes avant toute réduction et restitue la sortie brute si l'analyse est peu
fiable. Le projet n'est ni affilié à `rtk-ai`, ni approuvé par celui-ci.

La documentation française complète n'est pas encore disponible. Consultez
[README.md](README.md), [UPSTREAM.md](UPSTREAM.md) et
[docs/SAFETY_CONTRACT.md](docs/SAFETY_CONTRACT.md). N'utilisez pas les anciennes
instructions d'installation RTK pour ContextDroid.
