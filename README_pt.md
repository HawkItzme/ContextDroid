# ContextDroid

ContextDroid é um produto Android independente derivado do RTK. Ele salva stdout e stderr
brutos antes de reduzir a saída e retorna a saída original quando a confiança do analisador
é baixa. Não é afiliado nem endossado por `rtk-ai`.

A tradução completa ainda não está disponível. Consulte [README.md](README.md),
[UPSTREAM.md](UPSTREAM.md) e [docs/SAFETY_CONTRACT.md](docs/SAFETY_CONTRACT.md). Não use
as instruções antigas de instalação do RTK para ContextDroid.
